# AGENDS.md (Project Control File)

## Goal
Build an international care-introducer platform connecting families with verified carers and boutique micro-agencies. Focus on trust, transparency, and global adaptability.

## Core Features (MVP)
- User account creation (care seekers and carers)
- Care request posting and response
- Messaging between users
- Payment setup with multi-currency support
- Admin dashboard for basic management

## Security Rules
- Use Supabase backend with strict Row Level Security (RLS)
- Never expose API keys in frontend
- Validate all inputs on backend

## AI Workflow Preferences
- Use Claude for backend logic
- Use GPT-4 for frontend UI if needed
- Short, clear prompts
- Suggest safe and secure architecture

## Current Focus
Build “Request for Care” feature end-to-end
